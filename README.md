# FlashLearn
Flashlearn is a platform for generating flashcards from your own PDF.

## Features
- Generate flashcards from your own PDF
- Modify the flashcards
- Add to your collection
- Review your flashcards
- Track your progress


## Tech Stack
- Next.js App Router
- Tailwind CSS
- Shadcn UI
- Vercel AI SDK
- Better-Auth
- Postgresql
- Prisma ORM
